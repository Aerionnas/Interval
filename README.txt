This is an interval class and an interval test class. It has several methods such
as add, subtract, multiply, and average. The Test Interval class tests each method.

Project Title: Interval

Description: This program creates an Interval class that is able to perform all
of the operations possible.

Languages used: Java

For this project, I used two files: Interval.java and TestInterval.java. Please
find out more information below.

Interval.java- This is an Interval class that contains a blueprint for all
Interval Objects. You are also able to perform certain operations on these numbers.

TestInterval.java- This simply test the methods in the Interval class.

To run this project, compile the TestInterval.java file and run it in the
terminal.
